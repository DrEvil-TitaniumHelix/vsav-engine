Encoding a game needs four ingredients: the rulebook, the module's board and counters, a defined starting scenario, and charts or worked examples to validate against. For each game this page reports which ingredients are in hand and which are missing — and how to fix each gap. Essentially any board game can be encoded once its ingredients are supplied; the only thing this page judges is what we currently hold.
